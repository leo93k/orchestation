# PRD: 부동산 매물 서비스 (MVP)

## 목표

직방과 유사한 부동산 매물 조회 서비스의 MVP 개발

## MVP 범위

### 포함 (In Scope)

1. **매물 목록 조회** — 등록된 매물을 리스트로 조회
2. **매물 상세 보기** — 개별 매물의 상세 정보 확인 (사진, 가격, 면적, 위치, 설명)
3. **매물 필터링** — 지역, 거래 유형(전세/월세/매매), 가격대, 면적으로 필터
4. **매물 등록** — 관리자가 매물 정보를 등록/수정/삭제 (CRUD)

### 제외 (Out of Scope) — MVP 이후

- 지도 연동
- 실시간 채팅/문의
- 결제/중개 수수료
- 사용자 회원가입/로그인
- 찜/관심 매물
- 중개사 프로필

## 사용자 유형

| 유형        | 설명                                |
| ----------- | ----------------------------------- |
| 일반 사용자 | 매물 조회 및 필터링 (로그인 불필요) |
| 관리자      | 매물 CRUD                           |

## 핵심 화면

1. **매물 목록 페이지** — 필터 + 매물 카드 리스트
2. **매물 상세 페이지** — 상세 정보 표시
3. **관리자 페이지** — 매물 등록/수정/삭제

## 기술 스택

- 프론트엔드: Next.js (App Router)
- 백엔드: Node.js (Next.js Route Handler)
- DB: PostgreSQL + Prisma ORM
- 스타일링: Tailwind CSS
- 이미지: URL 입력 방식 (외부 업로드 없음)
- 관리자 인증: 없음 (MVP)

## 감독관 정책

- 과설계 금지: MVP에 없는 기능 추가 시 즉시 개입
- 각 기능은 독립적으로 Task 분리 가능해야 함
- 매물 데이터 모델을 먼저 확정한 후 UI/API 작업 시작
